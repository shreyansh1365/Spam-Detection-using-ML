# 1. INTRODUCTION (NETWORKING FUNDAMENTALS)

A computer network is a collection of interconnected devices that communicate using a set of standardized rules called **protocols**. These devices exchange data over logical communication endpoints known as **ports**.

In cybersecurity, understanding ports and protocols is critical because:

* Every attack targets a **protocol vulnerability**
* Every service runs on a **port**
* Network security is mainly about **controlling traffic between ports and protocols**

---

# 2. PORTS – COMPLETE THEORY

## 2.1 Definition

A **port** is a logical communication endpoint used by TCP/UDP protocols to identify specific services running on a system.

👉 Example:

* IP Address = Building address
* Port Number = Door of a specific service

---

## 2.2 Port Range Classification

| Range       | Type              | Description              |
| ----------- | ----------------- | ------------------------ |
| 0–1023      | Well-known ports  | Standard system services |
| 1024–49151  | Registered ports  | Application services     |
| 49152–65535 | Dynamic/Ephemeral | Temporary communication  |

---

## 2.3 Total Ports

* TCP Ports: 0–65535 → 65,536 ports
* UDP Ports: 0–65535 → 65,536 ports
* Total theoretical endpoints: 131,072 combinations

👉 But only ~50–70 ports are commonly used in real-world networking & cybersecurity.
